# Analysis of Online Shopping Store Annual Report

## Overview
An Excel-based comprehensive analysis of an online shopping store's annual performance, including sales trends, customer behavior, product performance, and revenue insights.

## Project Description
A detailed data analysis project that examines the annual operations of an online shopping store using Excel's powerful analytical tools and visualization features. The report provides actionable insights into sales patterns, customer demographics, product categories, and business performance metrics to support data-driven decision making.

## Features
- Sales trend analysis
- Customer demographics insights
- Product category performance
- Revenue and profit analysis
- Monthly/quarterly comparisons
- Interactive Excel dashboards
- Pivot tables and charts
- Key performance indicators (KPIs)

## Tools & Technologies
- Microsoft Excel (Advanced features)
- Pivot Tables
- Charts and Graphs
- Data Validation
- Conditional Formatting
- Excel Formulas and Functions

## Dataset Information
The analysis includes data on:
- Order details and transactions
- Customer information
- Product categories and inventory
- Sales revenue and profits
- Geographical distribution
- Time-based trends

## Key Metrics Analyzed

### Sales Analysis
- Total revenue

### Customer Insights

### Performance Indicators
- Profit margins

## Excel Features Used
- **Pivot Tables**: For data summarization
- **Charts**: Line, bar, pie charts for visualization
- **Conditional Formatting**: Highlighting key insights
- **Formulas**: SUMIF, VLOOKUP, INDEX-MATCH
- **Slicers**: Interactive filtering
- **Dashboards**: Visual summary of KPIs

## How to Use

1. **Open the Excel file**
   - Download the .xlsx file from the repository

2. **Explore the Dashboard**
   - Navigate to the Dashboard sheet
   - Use slicers to filter data
   - View key metrics and charts

3. **Analyze Data**
   - Review pivot tables for detailed insights
   - Check individual sheets for specific analysis
   - Customize filters as needed

4. **Update Data**
   - Add new data to the Raw_Data sheet
   - Refresh pivot tables (Alt + F5)
   - Dashboard updates automatically
